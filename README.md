## 📌 Overview
This repository contains a collection of Java practice programs focused on strengthening core programming concepts and logical thinking. The programs cover number-based logic, conditional statements, operators, switch-case implementation, character handling, and basic problem-solving techniques.

The objective of these exercises is to build a strong foundation in Java syntax and improve coding skills through hands-on implementation of real programming scenarios.

---

## 📚 Topics Covered
- Number manipulation programs
- Conditional statements (`if`, `if-else`, nested conditions)
- Switch case implementation
- Character and alphabet operations
- Ternary operator usage
- Arithmetic and logical operations
- Decision-making programs
- Basic mathematical logic building

---
